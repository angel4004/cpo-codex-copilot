# Skill candidates - кандидаты

Эта директория - staging area для workflows, которые позже могут стать реальными
Codex Skills. Файлы здесь не являются automatically discoverable runtime skills.

## Обновление PAF-канона

`paf-canon-updater` - controlled skill candidate для мониторинга
`productframework.ru`, новых PAF-ссылок и кандидатных внешних источников вроде
канала "Борода продакта". Он готовит proposal-first обновления канона, но не
делает внешние источники canonical authority без human review.
